 QUITAR CONTRASEÑA OLVIDADA DE INICIO DE SESIÓN DE WINDOWS

Hola amigos, aquí les tengo un pequeño tutorial con video para quitar contraseña de tu Windows sin formatear con una herramienta que todo Técnico debería de tener en sus manos Hiren’s BootCD PE 


Lo primero que tenemos que hacer es descargar Hiren’s BootCD PE para eso nos vamos a dirigir a su página principal para que lo descarguen abajo les dejo el link de su página web




¿Te preguntaras que es Hiren’s BootCD PE?

Hiren’s BootCD PE está basado en Windows 10 PE X64 y contiene una cantidad de herramientas gratuitas, pero las mejores para que puedas solucionar cualquier problema que se te presente en tu computadora y está desarrollada para computadoras de la nueva era, es compatible con arranque UEFI y requiere mínimo 2 GB de RAM.
 
Las Herramientas que contiene Hiren’s BootCD PE puedes solucionar muchos problemas que se te pueden presentar en tu computadora y no contienen ningún Software pirateado, solo contiene software gratuito.

Una vez descargado la ISO de Hiren’s BootCD PE tenemos que descargar y utilizar Rufus esta aplicación no va a hacer útil para hacer booteable nuestro pendrive y pueda inicar Hiren’s BootCD PE desde el pendrive 

¿Qué es Rufus?

Rufus es una pequeña aplicación que nos permite crear de forma segura un medio arranque boot en nuestras unidades USB de arranque, aquí te dejo el link para que descargue Rufus 




Ya que tengas la ISO de Hiren’s BootCD PE y Rufus vamos a hacer booteable nuestro Pendrive.

Tenemos que tener una memoria USB Como mínimo de 8 GB 


Conectamos nuestra memoria USB en tu Ordenador y ejecuta Rufus y selecciona el dispositivo USB que deseas configurar para instalar Hiren’s BootCD PE

En el campo "Elección de arranque", selecciona "Imagen ISO" y haz clic en el icono de disco para buscar y seleccionar la imagen ISO de Hiren’s BootCD PE descargada previamente.Y en el campo "Opciones de Imagen" vas a seleccionar Instalación de Windows Estándar

En el menú desplegable de "Formato del dispositivo", selecciona "Sistema de archivos: NTFS" si tu imagen ISO es mayor de 4GB y "Sistema de archivos: FAT32" si es menor a 4GB. En el campo de "Etiqueta del volumen" puedes colocarle el nombre que desees.

Haz clic en "Comenzar" y espera a que se complete el proceso de creación del USB de Hiren’s BootCD PE

Una vez que el proceso haya finalizado, podrás utilizar tu unidad USB para iniciar Hiren’s BootCD PE en cualquier ordenador compatible.

Conecta tu Pendrive en tu computadora, pero tenemos que configurar para que inicie desde nuestro pendrive, al momento de iniciar el equipo tenemos que seleccionar Generalmente las teclas (F2 o DEL) Para ingresar a la configuración de la BIOS y seleccionar como arranque principal nuestro pendrive. 

Si todo está perfecto tiene que iniciar perfectamente nuestro Hiren’s BootCD PE y como pueden ver es muy similar a Windows. 

En el escritorio vamos a buscar una carpeta que se llama Utilidades 

Luego damos clic en la carpeta que dice Security y veremos lo siguiente.

Y abrimos la carpeta Passwords y este será su contenido. 

Vamos a ejecutar la utilidad Lazesoft Windows Recovery



La vamos a dejar en la opción reset Windows password y luego presiona next 

Se nos va a abrir esta ventana y seleccionamos "YES"



Selecciona ahora Reset local password



Selecciona el Usuario que deseas desbloquear y presiona next



En la siguiente ventana vamos a seleccionar Reset Unlocked y debería de quitar nuestra contraseña de nuestro Windows, cerramos a la aplicación y vamos a iniciar con Windows para comprobar si quito la contraseña.

 



Efectivamente, ya ingresamos perfectamente a nuestro Windows sin la necesidad de ingresar nuestra contraseña. 

Como pueden ver gracias a Hiren’s BootCD PE es posible restablecer nuestra contraseña de nuestro Windows y así tener de nuevo acceso a nuestro sistema.

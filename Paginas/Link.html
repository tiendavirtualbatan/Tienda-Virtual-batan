Nuevos codigos

Pronto tendremos nuestro propio Hosting

Con precios comodos Factibles y muy seguro

Representación de plantillas HTML, actualización de bases de datos, envío y recepción de correos electrónicos, mantenimiento de páginas activas a través de WebSockets, puesta en cola de trabajos para trabajos asincrónicos, almacenamiento de cargas en la nube y provisión de sólidas protecciones de seguridad para ataques comunes. Rails lo hace todo y mucho más.

app/models/article.rb

Active Records facilita el modelado.

Las bases de datos cobran vida con la lógica empresarial encapsulada en objetos enriquecidos. Modelando asociaciones entre tablas, brindando devoluciones de llamada cuando se guardan, encriptando datos confidenciales sin problemas y expresando consultas SQL de manera hermosa.

app/controllers/articles_controller.rb

Los controladores de acción manejan todas las solicitudes.

Los controladores exponen el modelo de dominio a la web, procesan los parámetros entrantes, establecen encabezados de almacenamiento en caché y representan plantillas, respondiendo con HTML o JSON.

app/views/articles/show.html.erb

Las vistas de acción combinan Ruby y HTML.

Las plantillas pueden usar toda la versatilidad de Ruby, el código excesivo se extrae en ayudantes y el modelo de dominio se usa directamente y se entrelaza con el HTML.

config/routes.rb

URL de ruta de envío de acción.

Configure cómo se conectan las URL a los controladores utilizando el idioma del dominio de enrutamiento. Las rutas exponen el paquete de acciones que van juntas como un recurso: indexar, mostrar, nuevo, crear, editar, actualizar, destruir.
